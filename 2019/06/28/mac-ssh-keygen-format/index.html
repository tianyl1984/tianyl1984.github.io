开始结束标记中由 RSA 变成了 OPENSSH，导致部分软件如 DataGrip 之类通过 ssh 连接远程服务器识别秘钥失败，有两种方法可以解决：

方法一：

生成命令添加 -m PEM 指定格式为 PEM 格式

这样生成的私钥就是旧的格式，对于已有的私钥可以考虑方法二转换格式。

方法二：

安装 PuTTYgen，转换私钥格式。

安装方式参考官网：https://www.puttygen.com

id_sa 私钥文件
-O 大写英文字母O，指定输出格式
